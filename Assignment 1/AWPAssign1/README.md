# EventHub

EventHub is a React-based single page event management website built for Advance Web Programming Assignment 1. It allows users to explore upcoming events and register through a controlled form.

## Features

- React + Vite project setup
- Tailwind CSS styling
- Reusable functional components
- Events rendered from an array using props
- Controlled registration form using `useState`
- Responsive single-page layout

## Components

- Header
- AboutSection
- EventsSection
- EventCard
- RegistrationForm
- Footer

## Project Structure

```text
src/
  components/
  data/
  App.jsx
  main.jsx
  index.css
```


## Form Functionality

The registration form is a controlled form built with `useState`. It includes:

- Name field
- Email field
- Event selection dropdown
- Single input change handler
- Submit handling with displayed registration data
